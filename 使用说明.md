# AI小说写作助手 — 使用说明

## 简介

这是一个AI辅助写小说的工具。你只需要配置API Key，就可以与AI对话完成角色设定、情节构思、正文写作、修改润色，最后导出为txt/Word/PDF文件。

## 安装与启动

### 第一步：安装 Python
如果你还没有 Python，去 [python.org](https://python.org) 下载安装（版本 3.10 以上）。
安装时勾选 "Add Python to PATH"。

### 第二步：安装依赖
打开终端（Win+R → 输入 `cmd` → 回车），进入项目文件夹，运行：
```
pip install -r requirements.txt
```

### 第三步：启动
双击 `启动.bat`，浏览器打开 http://127.0.0.1:5000

## 首次使用

1. 注册账号（第一个注册的用户自动成为管理员）
2. 点右上角 ⚙️ 设置 → 填 API Key → 保存
3. 点 ✨ 新建 → 创建小说
4. 在左侧对话区与 AI 沟通

## 主要功能

| 功能 | 操作 |
|------|------|
| AI 写作 | 在对话区输入"开始写第一章"等 |
| 续写 | 点 ▶ 续写 按钮，AI 自动继续 |
| 修改 | 对话中说"把 XXX 改成 YYY"，AI 会输出修改后的全文 |
| 手动编辑 | 右侧面板点 ✏️编辑模式 |
| 角色管理 | 侧边栏 👥 手动编辑，或点 🎭 生成角色卡 让 AI 生成 |
| 情节大纲 | 侧边栏 📋 手动编辑，或点 📋 生成大纲 让 AI 生成 |
| 导出 | 点 📥 导出 → 选 txt / Word / PDF |
| 主题/字体 | ⚙️ 设置 → 外观标签页 |

## API Key 安全

你的 API Key 只存在你自己的浏览器里，不会上传到任何服务器。

## 多用户

- 每个人可以注册自己的账号
- 每个人的小说互相隔离，互不可见
- 管理员（首个注册用户）可以看到所有人的小说

## 常见问题

**Q: AI 拒绝写某些内容怎么办？**
A: 所有 AI 都有内容安全策略，对于敏感内容可能拒绝。尝试用更含蓄的表达方式。

**Q: 数据存在哪里？**
A: `data/` 文件夹。备份这个文件夹就可以迁移所有数据。

**Q: 能让朋友用吗？**
A: 可以，需要内网穿透工具（如 natapp、frp 等）把本地的 5000 端口暴露到公网。
